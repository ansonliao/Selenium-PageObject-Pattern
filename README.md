# Selenium-PageObject-Pattern

## Pre-Condition
Browsers:
Java version >= 8.0
Maven version >= 3.5

## Run
Run the test by command

```bash
mvn clean test -Dtest=MyTestRunner
```

Test will
1. run across `Chrome` , `Firefox`, and `PhantomJs`
2. download `chromddriver`, `geckodriver`, `phantomjsdriver` automatically
